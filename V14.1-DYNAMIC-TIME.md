# BLACKTERM OS v14.1 — Dynamic Time System

- Automatic dawn, day, dusk, night, and midnight visual profiles based on the visitor's local time.
- Scene brightness, saturation, particles, HUD status, and event intensity adapt by phase.
- Midnight Protocol introduces rare transmissions and stronger atmospheric effects from 12:00 AM–2:59 AM.
- Toggle from Settings → Effects → Dynamic day / night.
