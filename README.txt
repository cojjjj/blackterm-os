BLACKTERM OS v13.5 — Cinematic Desktop Environments

Copy the included src folder into the root of your current BLACKTERM OS project and allow it to replace matching files.

Then run:
  npm.cmd run build
  git add .
  git commit -m "Add cinematic reactive desktop environments"
  git push

Features:
- Ten distinct live desktop scenes
- Mouse-reactive parallax layers
- Fog, rain, particles, radar, orbit rings, beams, lightning, and glitch events
- Wallpaper-specific motion profiles
- Active-app reactions for Terminal, Threat Map, Incident Engine, Sandbox, Assistant, and Resume
- Reduced-motion accessibility support
